Picolo is a simple libary for generating all the data prescribed in an image: The dimensions of the image and all pixels

Example

Basic usage for reading and translating a complete Image and translating it to a comprehensive table:

let pixl_struct = load_picture("images/ig-icon.png");

for i in pixl_struct {
    println!("{:?}", i); 
}
